## 🌟 Featured Projects

### **Stagehand**
**Multi-Agent Workflow Builder with Live Execution Tracing**

A full-stack platform for designing and running agent workflows on a drag-and-drop canvas, with live execution visibility, replay, and debugging.

**Tech:** React Flow, FastAPI, PostgreSQL, ClickHouse, WebSockets, LLMs

---

### **Forecast Studio**
**Natural-Language Analytics with SQL Verification**

A full-stack app that turns plain-English questions into verified SQL against Postgres or Snowflake, then checks whether the result actually answers the original question.

**Tech:** Text-to-SQL, FastAPI, PostgreSQL, Snowflake, LLMs

---

### **Onboard.dev**
**AI Codebase Onboarding with Dependency Graph Analysis**

A developer tool that ingests GitHub repositories, maps dependencies across modules, ranks what matters most, and generates guided learning with line-level code citations.

**Tech:** Next.js, RAG, pgvector, D3, FastAPI

---

### **SolHive**

A blockchain analytics project built during HackNYU, focused on turning raw decentralized transaction activity into useful insights.

**Tech:** Blockchain, Python, APIs, Data Visualization
